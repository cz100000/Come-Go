Arbeitszeit-PWA

1. Den Ordner auf einen kleinen HTTPS-Webspace hochladen (z. B. GitHub Pages).
2. index.html in Safari auf dem iPhone öffnen.
3. Teilen > Zum Home-Bildschirm.

Für eine reine lokale Desktop-Nutzung kann index.html direkt geöffnet werden. Die PWA-Installation und der Service Worker benötigen HTTPS oder localhost.

Die Daten liegen lokal im Browser. Regelmäßig unter Einstellungen ein JSON-Backup exportieren.
