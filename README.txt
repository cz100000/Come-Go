Arbeitszeit-PWA – Version 2

1. Den Ordner auf einen HTTPS-Webspace hochladen (z. B. GitHub Pages).
2. index.html in Safari auf dem iPhone öffnen.
3. Teilen > Zum Home-Bildschirm.

Neu in Version 2:
- fortlaufendes Überstunden-/Unterstundenkonto
- Übertrag zwischen Monaten und Jahren
- einstellbarer Anfangssaldo
- Zeitkontostand in Tages-, Monats- und Jahresansicht
- zukünftige Tage erzeugen noch keine Minusstunden

Die Daten liegen lokal im Browser. Regelmäßig unter Einstellungen ein JSON-Backup exportieren.
